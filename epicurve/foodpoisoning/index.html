Food Poisoning Epidemic Curve Generator

Epidemic Curve Generator for Food Poisoning
Easily generate an epidemic curve from Food Poisoning linelisting extracted from eNotifikasi online system.

Data requirement:
This app uses the following eNotis fields:

  • Epid Tahun (Tkh Onset)
  • Epid Minggu (Tkh Onset)
All processing is done locally in the browser.
